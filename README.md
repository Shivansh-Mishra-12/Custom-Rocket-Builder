# 🚀 Custom Rocket Builder

An interactive 3D web experience where users can explore, select, highlight, and interact with different rocket models directly in the browser.

**Live Demo:** [custom-rocket-builder.vercel.app](https://custom-rocket-builder.vercel.app/)

---

## Overview

Custom Rocket Builder is a 3D configurator-style website built for rocket enthusiasts and curious minds. It renders multiple rocket models in a real-time 3D environment, letting users select and highlight different rockets as a foundation for a richer configurator experience.

---

## Features

- 🛸 **Multiple Rocket Models** — Browse a collection of distinct rocket designs in a shared 3D scene
- 🖱️ **Select & Highlight** — Click to select and visually highlight a rocket model in real time

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React + Vite |
| 3D Rendering | Three.js + React Three Fiber |
| 3D Helpers | Drei (`@react-three/drei`) |
| Deployment | Vercel |

## Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/shivansh-mishra-12/custom-rocket-builder.git
cd custom-rocket-builder

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
npm run preview
```
